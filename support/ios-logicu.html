LogICU FAQ

Frequently asked questions and support information

Import / Export

How do I import my backup record?
  1. Send your backup CSV file to your iPhone through email. Right now, only LogICU backup CSV files are supported.
  2. Click the CSV file and select "Open in LogICU"
  3. When you import the record, you can select to keep or delete all the records already stored in the device
How do I export my record?

Go to Settings → Export CSV via Email

Configuration

How do I configure the report header?

Go to Settings → User section: input your name, grade and trust

How do I configure the quick search?

Go to Settings → Search section: select the keyword you want to use for quick search. Right now, Involvement, Assessment, Diagnosis, Comments, Patient ID and Hospital ID are supported.

How do I configure the keyword suggestion?

Go to Settings → Data section
